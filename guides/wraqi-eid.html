Comment utiliser Wraqi et Mon e-ID du Maroc

Les services électroniques de l'administration marocaine vous permettent désormais de demander et de consulter divers documents administratifs sans vous déplacer. Ce guide vous explique comment utiliser Wraqi pour vos documents et Mon e-ID pour votre identité numérique au Maroc.

Étape 1 : Comprendre Wraqi et mon e-ID

Qu'est-ce que Wraqi ?

  • Wraqi est une plateforme digitale développée par l'Agence de Développement du Digital (ADD) au Maroc
  • Elle permet de demander, télécharger et vérifier des documents administratifs en ligne
  • La plateforme vise à simplifier les démarches administratives et à réduire l'utilisation du papier

Qu'est-ce que Mon e-ID ?

  • Mon e-ID est l'identité numérique officielle des citoyens marocains
  • Elle permet de s'authentifier de manière sécurisée sur les plateformes gouvernementales
  • Cette identité numérique facilite l'accès aux services administratifs en ligne

Étape 2 : Créer un compte Mon e-ID

Sur ordinateur :

  1. Accéder à la plateforme e-ID :
    • Ouvrez votre navigateur internet et allez sur mon-eid.ma
    • Cliquez sur "Créer un compte"
  2. Remplir le formulaire d'inscription :
    • Saisissez votre Carte Nationale d'Identité Électronique (CNIE) au format "AB123456"
    • Entrez votre numéro de téléphone mobile (qui doit être enregistré à votre nom)
    • Sélectionnez votre date de naissance
    • Cochez la case pour accepter les conditions d'utilisation
    • Cliquez sur "Suivant"
  3. Vérifier votre identité :
    • Un code de vérification sera envoyé par SMS à votre numéro
    • Entrez ce code dans le champ approprié
    • Cliquez sur "Vérifier"
  4. Compléter votre profil :
    • Créez un mot de passe fort (combinaison de lettres, chiffres et caractères spéciaux)
    • Entrez une adresse e-mail valide
    • Choisissez des questions de sécurité et fournissez les réponses
    • Cliquez sur "Créer mon compte"

Sur smartphone ou tablette :

  1. Télécharger l'application :
    • Ouvrez le Play Store (Android) ou l'App Store (iPhone/iPad)
    • Recherchez "Mon e-ID Maroc"
    • Appuyez sur "Installer" ou "Obtenir"
    • Une fois installée, ouvrez l'application
  2. Créer un compte :
    • Appuyez sur "Créer un compte"
    • Suivez les mêmes étapes que sur ordinateur pour remplir le formulaire
    • L'application vous demandera d'autoriser l'accès à votre appareil photo (pour scanner votre CNIE si nécessaire)

Étape 3 : Se connecter et utiliser Mon e-ID

Connexion à votre compte :

  1. Sur ordinateur :
    • Allez sur mon-eid.ma
    • Entrez votre identifiant (numéro CNIE) et votre mot de passe
    • Cliquez sur "Se connecter"
    • Un code de vérification sera envoyé par SMS
    • Entrez ce code pour finaliser la connexion
  2. Sur l'application mobile :
    • Ouvrez l'application Mon e-ID
    • Entrez votre identifiant et mot de passe
    • Appuyez sur "Se connecter"
    • Entrez le code de vérification reçu par SMS

Fonctionnalités principales de Mon e-ID :

  1. Consulter vos informations :
    • Dans le tableau de bord, vous pouvez voir vos informations personnelles
    • Vérifiez que toutes les informations sont correctes
  2. Modifier vos informations de contact :
    • Si nécessaire, mettez à jour votre adresse e-mail ou numéro de téléphone
    • Cliquez sur "Modifier" à côté de l'information à mettre à jour
    • Suivez les instructions pour vérifier et valider les modifications

Étape 4 : Utiliser Wraqi pour demander des documents administratifs

Accéder à Wraqi :

  1. Sur ordinateur :
    • Allez sur wraqi.ma
    • Cliquez sur "Se connecter avec Mon e-ID"
    • Vous serez redirigé vers la page de connexion de Mon e-ID
    • Connectez-vous avec vos identifiants Mon e-ID
  2. Sur l'application mobile :
    • Téléchargez l'application "Wraqi" depuis le Play Store ou l'App Store
    • Ouvrez l'application et appuyez sur "Se connecter avec Mon e-ID"
    • Suivez le processus d'authentification

Demander un document administratif :

  1. Sélectionner le type de document :
    • Sur la page d'accueil de Wraqi, vous verrez les différents types de documents disponibles
    • Les documents généralement disponibles incluent :
      • Extrait d'acte de naissance
      • Certificat de résidence
      • Certificat de vie
      • Extrait du casier judiciaire
    • Cliquez ou appuyez sur le document souhaité
  2. Remplir le formulaire de demande :
    • Un formulaire spécifique au document s'affichera
    • Certains champs seront déjà remplis avec vos informations personnelles
    • Complétez les champs manquants selon les instructions
    • Si nécessaire, téléchargez les pièces justificatives demandées
  3. Soumettre votre demande :
    • Vérifiez que toutes les informations sont correctes
    • Cliquez sur "Soumettre" ou "Envoyer la demande"
    • Vous recevrez une confirmation avec un numéro de référence
    • Conservez ce numéro pour suivre l'état de votre demande

Étape 5 : Suivre et récupérer vos documents

Suivi de vos demandes :

  1. Consulter l'état de vos demandes :
    • Dans Wraqi, accédez à la section "Mes demandes" ou "Suivi des demandes"
    • Vous verrez la liste de toutes vos demandes en cours et passées
    • L'état de chaque demande sera indiqué (En cours de traitement, Validé, Refusé, etc.)
  2. Notifications :
    • Vous recevrez des notifications par SMS et/ou e-mail concernant l'état de vos demandes
    • Ces notifications vous informeront quand votre document est prêt

Télécharger vos documents :

  1. Documents électroniques :
    • Lorsqu'un document est prêt, vous pouvez le télécharger directement depuis Wraqi
    • Dans la section "Mes demandes", cliquez sur "Télécharger" à côté du document validé
    • Le document sera au format PDF et comportera un QR code pour vérification
    • Vous pouvez imprimer ce document ou le conserver au format numérique selon vos besoins
  2. Documents physiques :
    • Certains documents peuvent nécessiter un retrait en personne
    • Dans ce cas, Wraqi vous indiquera où et quand vous pouvez récupérer le document
    • N'oubliez pas d'apporter votre CNIE et la référence de votre demande

Conseils pour utiliser Wraqi et Mon e-ID en toute sécurité :

  • Protégez vos identifiants : Ne partagez jamais vos identifiants Mon e-ID avec qui que ce soit
  • Utilisez un mot de passe fort : Créez un mot de passe unique et complexe pour votre compte
  • Vérifiez l'URL : Assurez-vous toujours que vous êtes sur le site officiel avant de vous connecter (wraqi.ma ou mon-eid.ma)
  • Mettez à jour vos coordonnées : Gardez votre numéro de téléphone et votre email à jour pour recevoir les codes de vérification
  • Déconnectez-vous après utilisation : Surtout si vous utilisez un ordinateur partagé ou public
  • Vérifiez régulièrement vos demandes : Consultez l'état de vos demandes pour ne pas manquer une notification
  • Conservez les références : Notez les numéros de référence de vos demandes dans un endroit sûr

Note importante : Les documents administratifs obtenus via Wraqi sont officiels et légalement valides. Ils comportent des éléments de sécurité (QR code, signature électronique) qui permettent de vérifier leur authenticité.
