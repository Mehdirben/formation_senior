Comment utiliser YouTube : fonctions avancées

YouTube est une plateforme qui vous permet non seulement de regarder des vidéos, mais aussi de les organiser, de créer vos propres playlists et de partager du contenu avec vos proches. Ce guide vous explique comment utiliser YouTube de manière plus avancée sur ordinateur et sur mobile.

Étape 1 : Créer un compte YouTube (si vous n'en avez pas déjà un)

Sur ordinateur :

  1. Accéder à YouTube :
    • Ouvrez votre navigateur internet
    • Tapez www.youtube.com dans la barre d'adresse
  2. Se connecter ou créer un compte :
    • Cliquez sur "Connexion" en haut à droite de l'écran
    • Si vous avez déjà un compte Google (Gmail), entrez vos identifiants
    • Sinon, cliquez sur "Créer un compte" et suivez les instructions
    • Un compte YouTube est en fait un compte Google, vous pouvez donc utiliser la même adresse email et le même mot de passe que pour Gmail

Sur smartphone ou tablette :

  1. Télécharger l'application YouTube :
    • Ouvrez le Play Store (Android) ou l'App Store (iPhone/iPad)
    • Recherchez "YouTube" et appuyez sur "Installer" ou "Obtenir"
    • Une fois installée, ouvrez l'application
  2. Se connecter :
    • Appuyez sur l'icône de profil en haut à droite
    • Appuyez sur "Se connecter"
    • Suivez les instructions pour vous connecter avec votre compte Google ou en créer un nouveau

Étape 2 : Créer des playlists personnalisées

Sur ordinateur :

  1. Créer une nouvelle playlist :
    • Trouvez une vidéo que vous souhaitez ajouter à une playlist
    • Sous la vidéo, cliquez sur "Enregistrer" (icône représentant une liste avec un signe "+")
    • Dans le menu qui s'ouvre, cliquez sur "Créer une nouvelle playlist"
    • Donnez un nom à votre playlist
    • Choisissez la confidentialité de la playlist : Publique (visible par tous), Non répertoriée (accessible uniquement via un lien) ou Privée (visible uniquement par vous)
    • Cliquez sur "Créer"
  2. Ajouter d'autres vidéos à une playlist existante :
    • Pour toute autre vidéo que vous souhaitez ajouter, cliquez sur "Enregistrer" sous la vidéo
    • Cochez la case à côté du nom de votre playlist
    • La vidéo sera automatiquement ajoutée à cette playlist

Sur smartphone ou tablette :

  1. Créer une nouvelle playlist :
    • Trouvez une vidéo que vous souhaitez sauvegarder
    • Appuyez sur "Enregistrer" (icône de liste sous la vidéo ou dans le menu à trois points)
    • Appuyez sur "+ Nouvelle playlist"
    • Donnez un nom à votre playlist
    • Choisissez les paramètres de confidentialité
    • Appuyez sur "Créer"
  2. Ajouter à une playlist existante :
    • Lorsque vous trouvez une vidéo que vous souhaitez ajouter, appuyez sur "Enregistrer"
    • Sélectionnez la playlist de votre choix dans la liste

Étape 3 : Gérer et modifier vos playlists

Sur ordinateur :

  1. Accéder à vos playlists :
    • Cliquez sur le menu à gauche de l'écran
    • Cliquez sur "Bibliothèque" ou "Playlists"
    • Vous verrez toutes vos playlists créées ou sauvegardées
  2. Modifier une playlist :
    • Cliquez sur la playlist que vous souhaitez modifier
    • Pour changer le titre ou la confidentialité : cliquez sur les trois points (...) à côté du titre, puis sur "Modifier"
    • Pour réorganiser les vidéos : cliquez et maintenez sur les lignes à côté d'une vidéo, puis faites-la glisser vers sa nouvelle position
    • Pour supprimer une vidéo : cliquez sur les trois points à côté de la vidéo, puis sur "Supprimer de [nom de la playlist]"
  3. Supprimer une playlist entière :
    • Allez sur la page de la playlist
    • Cliquez sur les trois points (...) à côté du nom de la playlist
    • Sélectionnez "Supprimer la playlist"
    • Confirmez la suppression

Sur smartphone ou tablette :

  1. Accéder à vos playlists :
    • Appuyez sur "Bibliothèque" en bas de l'écran
    • Faites défiler jusqu'à la section "Playlists"
    • Appuyez sur "Voir tout" pour afficher toutes vos playlists
  2. Modifier une playlist :
    • Ouvrez la playlist que vous souhaitez modifier
    • Appuyez sur l'icône de crayon ou sur les trois points en haut de l'écran
    • Pour changer le titre ou la confidentialité : appuyez sur "Modifier"
    • Pour réorganiser : maintenez et faites glisser les vidéos en utilisant les lignes à droite
    • Pour supprimer : appuyez sur les "X" à côté des vidéos ou utilisez le menu à trois points

Étape 4 : Partager des playlists et des vidéos

Sur ordinateur :

  1. Partager une vidéo :
    • Sous la vidéo que vous regardez, cliquez sur "Partager" (icône en forme de flèche)
    • Plusieurs options s'offrent à vous : par email, sur les réseaux sociaux, ou en copiant le lien
    • Pour copier le lien, cliquez sur "Copier" et collez-le ensuite dans un email ou un message
    • Pour partager à partir d'un moment précis de la vidéo, cochez "Commencer à" et ajustez le temps
  2. Partager une playlist entière :
    • Ouvrez la playlist que vous souhaitez partager
    • Cliquez sur "Partager" sous le titre de la playlist
    • Choisissez votre méthode de partage ou copiez le lien
    • Important : Seules les playlists publiques ou non répertoriées peuvent être partagées. Les playlists privées ne sont visibles que par vous

Sur smartphone ou tablette :

  1. Partager une vidéo :
    • Pendant la lecture d'une vidéo, appuyez sur "Partager" (l'icône de flèche)
    • Ou appuyez sur les trois points sous la vidéo et sélectionnez "Partager"
    • Choisissez l'application par laquelle vous souhaitez partager (WhatsApp, Email, Messages, etc.)
  2. Partager une playlist :
    • Ouvrez la playlist
    • Appuyez sur les trois points en haut ou sur l'icône de partage
    • Sélectionnez l'application pour partager

Étape 5 : Personnaliser votre expérience YouTube

S'abonner à des chaînes :

  1. Sur ordinateur :
    • Allez sur la page d'une chaîne en cliquant sur le nom de la chaîne sous une vidéo
    • Cliquez sur le bouton rouge "S'abonner"
    • Vous pouvez activer la cloche de notification pour être alerté des nouvelles vidéos
  2. Sur mobile :
    • Appuyez sur le nom de la chaîne sous une vidéo
    • Sur la page de la chaîne, appuyez sur "S'abonner"
    • Appuyez sur la cloche pour personnaliser vos notifications

Gérer votre historique de visionnage :

  1. Sur ordinateur :
    • Dans le menu de gauche, cliquez sur "Historique"
    • Pour supprimer une vidéo de l'historique : placez le curseur sur la vidéo et cliquez sur "X"
    • Pour effacer tout l'historique : dans le menu de droite, cliquez sur "Effacer tout l'historique"
    • Pour mettre en pause l'enregistrement de l'historique : cliquez sur "Suspendre l'historique"
  2. Sur mobile :
    • Appuyez sur "Bibliothèque" puis sur "Historique"
    • Pour supprimer des vidéos : appuyez sur les trois points à côté d'une vidéo puis sur "Supprimer de l'historique"
    • Pour gérer l'historique : appuyez sur les trois points en haut à droite puis sur "Paramètres d'historique"

Étape 6 : Fonctionnalités utiles pour les seniors

Modifier la taille de la vidéo et les sous-titres :

  1. Ajuster la taille de la vidéo :
    • Sur ordinateur : cliquez sur l'icône "Plein écran" en bas à droite du lecteur vidéo
    • Sur mobile : tournez votre appareil horizontalement pour un affichage plus grand
    • Sur les Smart TV : de nombreuses Smart TV ont l'application YouTube préinstallée pour une meilleure expérience sur grand écran
  2. Activer les sous-titres :
    • Sur ordinateur : cliquez sur l'icône "CC" en bas du lecteur vidéo
    • Sur mobile : appuyez sur les trois points pendant la lecture puis sur "Sous-titres/CC"
    • Vous pouvez également ajuster la taille et la couleur des sous-titres dans les paramètres

Utiliser YouTube sur votre téléviseur :

  1. Via une Smart TV :
    • Si vous avez une Smart TV, cherchez l'application YouTube dans le magasin d'applications
    • Installez-la si elle n'est pas déjà présente
    • Lancez l'application et connectez-vous à votre compte
  2. Via Chromecast ou appareil similaire :
    • Si vous avez un Chromecast branché à votre téléviseur :
      • Sur mobile : pendant la lecture d'une vidéo, appuyez sur l'icône de diffusion (ressemble à un écran avec des ondes wifi)
      • Sur ordinateur : cliquez sur les trois points dans le coin inférieur droit du lecteur, puis sur "Diffuser"
    • Sélectionnez votre Chromecast ou appareil compatible dans la liste
    • La vidéo sera alors diffusée sur votre téléviseur

Conseils pour une meilleure expérience sur YouTube :

  • Utilisez le mode sombre pour moins fatiguer vos yeux :
    • Sur ordinateur : cliquez sur votre photo de profil > Apparence > Thème sombre
    • Sur mobile : appuyez sur votre photo de profil > Paramètres > Général > Thème sombre
  • Réglez la qualité vidéo selon votre connexion internet :
    • Pendant la lecture d'une vidéo, cliquez/appuyez sur l'icône d'engrenage
    • Sélectionnez "Qualité" puis choisissez la résolution appropriée
    • Une qualité plus basse (480p ou moins) est préférable si votre connexion est lente
  • Utilisez la fonction "A regarder plus tard" pour marquer les vidéos que vous voulez voir ultérieurement :
    • Cliquez/appuyez sur "Enregistrer" sous une vidéo
    • Sélectionnez "À regarder plus tard"
    • Vous retrouverez ces vidéos dans votre Bibliothèque > À regarder plus tard
  • Créez des playlists thématiques (recettes de cuisine, jardinage, exercices physiques, etc.) pour organiser vos centres d'intérêt
  • Explorez les catégories populaires comme Musique, Actualités, ou Éducation pour découvrir de nouveaux contenus

Note importante : YouTube met régulièrement à jour son interface. Si certains boutons ou fonctionnalités ne se trouvent pas exactement où ils sont décrits dans ce guide, explorez les menus et les icônes pour les trouver. Les fonctionnalités principales restent généralement similaires, même si leur emplacement peut changer.
