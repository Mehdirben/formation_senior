Applications de transport

Les applications de transport vous permettent de commander facilement des taxis, des VTC (Voitures de Transport avec Chauffeur) ou de consulter les horaires des transports en commun. Ce guide vous explique comment utiliser les applications de transport les plus populaires.

Étape 1 : Choisir et installer une application de transport

Applications de VTC et taxis :

  1. InDrive (disponible dans de nombreuses villes) :
    • Ouvrez le Play Store (Android) ou l'App Store (iPhone/iPad)
    • Dans la barre de recherche, tapez "InDrive"
    • Appuyez sur "Installer" ou "Obtenir", puis attendez la fin de l'installation
  2. Autres options selon votre région :
    • Uber : très répandu dans la plupart des grandes villes
    • Careem : disponible dans certaines régions

Étape 2 : Configurer votre compte InDrive

Première inscription :

  1. Ouvrir l'application :
    • Localisez l'icône InDrive sur votre écran d'accueil et appuyez dessus
  2. Créer un compte :
    • Sélectionnez votre pays et entrez votre numéro de téléphone
    • Un code de vérification sera envoyé par SMS à ce numéro
    • Entrez le code reçu dans l'application
    • Complétez vos informations personnelles (nom et prénom)
    • Acceptez les conditions générales après les avoir lues
  3. Configurer votre profil :
    • Vous pouvez ajouter une photo de profil (optionnel)
    • Vérifiez que vos informations sont correctes

Étape 3 : Commander une voiture avec InDrive

Sur l'application InDrive :

  1. Indiquer vos points de départ et d'arrivée :
    • Sur l'écran principal, appuyez sur le champ "D'où"
    • Entrez votre adresse de départ ou utilisez votre position actuelle
    • Appuyez ensuite sur "Où" et entrez votre destination
    • Vous pouvez sélectionner des lieux sur la carte ou utiliser la recherche
  2. Proposer votre prix (caractéristique unique d'InDrive) :
    • InDrive vous suggère un prix de base pour votre trajet
    • Vous pouvez soit accepter ce prix, soit proposer votre propre montant
    • Utilisez le curseur ou entrez manuellement le montant que vous souhaitez payer
    • Sachez que proposer un prix trop bas peut réduire vos chances de trouver un chauffeur
  3. Choisir le type de service :
    • Sélectionnez le type de véhicule souhaité : standard, confort, ou autres options disponibles
    • Vous pouvez ajouter des informations supplémentaires pour le chauffeur
  4. Commander votre trajet :
    • Appuyez sur "Commander" ou "Envoyer une demande"
    • Votre demande sera visible par les chauffeurs disponibles dans votre zone
    • Les chauffeurs peuvent accepter votre offre ou faire une contre-proposition
    • Vous recevrez des notifications lorsque des chauffeurs répondront à votre demande
  5. Sélectionner un chauffeur :
    • Consultez les profils des chauffeurs qui ont accepté votre demande
    • Vérifiez leur note, leur véhicule et leurs avis
    • Choisissez le chauffeur qui vous convient le mieux

Étape 4 : Pendant et après votre trajet InDrive

Pendant votre trajet :

  1. Suivre votre trajet :
    • L'application affiche la position du chauffeur qui vient vous chercher
    • Vous pouvez communiquer avec le chauffeur par chat ou appel téléphonique via l'application
    • Pendant le trajet, vous pouvez suivre votre parcours sur la carte
  2. Fonctionnalités de sécurité :
    • Utilisez le bouton SOS en cas d'urgence
    • Vous pouvez partager les détails de votre trajet avec un proche

À la fin de votre trajet :

  1. Paiement :
    • Contrairement à d'autres applications, InDrive fonctionne principalement avec des paiements en espèces
    • Payez directement au chauffeur le montant convenu
    • Dans certaines régions, le paiement par carte peut être disponible
  2. Évaluation :
    • Après le trajet, évaluez votre expérience en attribuant une note au chauffeur
    • Vous pouvez également laisser un commentaire sur la qualité du service
    • Ces évaluations aident les autres utilisateurs à choisir leurs chauffeurs
  3. Objets oubliés :
    • Si vous avez oublié quelque chose dans le véhicule, contactez rapidement le chauffeur via l'application
    • Vous pouvez également contacter le support client d'InDrive pour obtenir de l'aide

Conseils pour bien utiliser les applications de transport :

  • Vérifiez votre batterie : Assurez-vous que votre téléphone est suffisamment chargé avant de commander un VTC ou de partir en transport en commun
  • Proposez un prix juste sur InDrive : Un prix trop bas pourrait dissuader les chauffeurs, mais vous pouvez négocier intelligemment
  • Comparez les prix : Installez plusieurs applications de VTC pour comparer les prix et les temps d'attente
  • Planifiez à l'avance : Pour les trajets importants, commandez votre voiture avec suffisamment d'avance
  • Téléchargez les plans de transports : Dans Moovit, téléchargez les plans pour une utilisation hors ligne
  • Consultez les avis : Avant d'utiliser une nouvelle application de transport, regardez les avis d'autres utilisateurs
  • Préparez votre monnaie : Avec InDrive, assurez-vous d'avoir suffisamment d'espèces pour payer votre course

Note importante : Pour votre sécurité, vérifiez toujours l'identité du chauffeur et la plaque d'immatriculation du véhicule avant de monter dans un VTC. Ne montez jamais dans un véhicule dont les informations ne correspondent pas à celles affichées dans l'application.
