Comment sauvegarder WhatsApp

WhatsApp est une application de messagerie populaire qui vous permet de communiquer avec vos proches. Il est important de sauvegarder régulièrement vos conversations et médias pour ne pas les perdre en cas de changement de téléphone ou de problème technique. Ce guide vous explique comment faire.

Sauvegarde WhatsApp sur Android

Utiliser Google Drive pour sauvegarder automatiquement :

  1. Ouvrez WhatsApp sur votre téléphone Android
  2. Appuyez sur les trois points en haut à droite pour ouvrir le menu
  3. Sélectionnez Paramètres
  4. Appuyez sur Discussions
  5. Sélectionnez Sauvegarde des discussions
  6. Choisissez le compte Google sur lequel vous souhaitez sauvegarder vos données
  7. Définissez la fréquence de sauvegarde (quotidienne, hebdomadaire, mensuelle ou jamais)
  8. Choisissez si vous voulez inclure les vidéos dans votre sauvegarde (attention, cela prend plus d'espace)
  9. Sélectionnez si vous préférez sauvegarder uniquement via Wi-Fi ou également via les données mobiles
  10. Appuyez sur SAUVEGARDER pour lancer une sauvegarde manuelle immédiate

Sauvegarde WhatsApp sur iPhone

Utiliser iCloud pour sauvegarder automatiquement :

  1. Ouvrez WhatsApp sur votre iPhone
  2. Appuyez sur Réglages en bas à droite de l'écran
  3. Sélectionnez Discussions
  4. Appuyez sur Sauvegarde des discussions
  5. Définissez la fréquence de sauvegarde (automatique, quotidienne, hebdomadaire, mensuelle ou jamais)
  6. Choisissez si vous voulez inclure les vidéos dans votre sauvegarde
  7. Appuyez sur Sauvegarder maintenant pour lancer une sauvegarde manuelle immédiate
  8. Important : Vérifiez que la fonction iCloud Drive est activée sur votre iPhone (dans Réglages > [votre nom] > iCloud > iCloud Drive)

Restaurer une sauvegarde WhatsApp

Sur un nouveau téléphone ou après réinstallation :

  1. Installez WhatsApp sur votre nouveau téléphone
  2. Vérifiez que vous utilisez le même numéro de téléphone que celui utilisé pour la sauvegarde
  3. Lors de la configuration initiale, WhatsApp vous proposera automatiquement de restaurer votre sauvegarde s'il en détecte une
  4. Suivez les instructions à l'écran pour restaurer vos conversations et médias
  5. Le processus peut prendre du temps selon la taille de votre sauvegarde et la vitesse de votre connexion internet

Exportation manuelle de conversations importantes

Pour conserver certaines conversations spécifiques :

  1. Ouvrez la conversation que vous souhaitez exporter dans WhatsApp
  2. Appuyez sur les trois points en haut à droite (Android) ou sur le nom du contact en haut (iPhone)
  3. Sur Android :
    • Sélectionnez Plus
    • Appuyez sur Exporter la discussion
    • Choisissez si vous voulez inclure les médias ou non
    • Sélectionnez l'application avec laquelle vous souhaitez partager la conversation (Gmail, Drive, etc.)
  4. Sur iPhone :
    • Faites défiler vers le bas et appuyez sur Exporter la discussion
    • Choisissez si vous voulez inclure les médias ou non
    • Sélectionnez la méthode de partage (Mail, Notes, etc.)
  5. La conversation sera exportée sous forme de fichier texte (.txt) avec les médias joints si vous avez choisi cette option

Conseils pour une sauvegarde efficace :

  • Sauvegardez régulièrement : Configurez des sauvegardes automatiques pour ne pas avoir à y penser
  • Vérifiez l'espace disponible : Assurez-vous d'avoir suffisamment d'espace sur votre compte Google Drive ou iCloud
  • Connexion Wi-Fi : Effectuez les sauvegardes sur Wi-Fi pour économiser vos données mobiles
  • Temps de sauvegarde : Les premières sauvegardes peuvent prendre beaucoup de temps. Faites-les de préférence la nuit
  • Médias importants : Pensez à sauvegarder séparément les photos et vidéos importantes en les enregistrant dans votre galerie
  • Changement de plateforme : Sachez qu'il n'est pas possible de transférer directement une sauvegarde entre Android et iPhone (il existe des applications tierces pour cela, mais elles ne sont pas officielles)

Note importante : Les sauvegardes de WhatsApp comprennent vos messages et les médias partagés, mais pas vos appels audio et vidéo. Vos conversations sont chiffrées dans la sauvegarde pour protéger votre vie privée.
